API Endpoints

GET /api/getnumber

Get next available number from pool (removes it)

curl http://localhost:5000/api/getnumber
POST /api/numbers/upload

Upload numbers from text file

curl -X POST http://localhost:5000/api/numbers/upload -F "file=@numbers.txt"
GET /api/numbers/pool

Get number pool status

curl http://localhost:5000/api/numbers/pool
GET /api/otp/{number}
curl http://localhost:5000/api/otp/1234567890
POST /api/otp
curl -X POST http://localhost:5000/api/otp -H "Content-Type: application/json" -d '{"number":"1234567890"}'
GET /api/stats
