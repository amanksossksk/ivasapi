API Endpoints

GET /api/getnumber

Get next available number from current pool

curl http://localhost:5000/api/getnumber
GET /api/pools

Get all pools information

curl http://localhost:5000/api/pools
POST /api/pools/switch

Switch to a different pool

curl -X POST http://localhost:5000/api/pools/switch -H "Content-Type: application/json" -d '{"pool":"guatemala"}'
POST /api/pools/create

Create a new pool

curl -X POST http://localhost:5000/api/pools/create -H "Content-Type: application/json" -d '{"name":"my_pool","description":"My custom pool","country":"USA"}'
